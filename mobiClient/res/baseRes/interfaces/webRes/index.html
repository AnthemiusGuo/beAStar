规则说明

斗地主起源于湖北，该游戏由三个人用一副54 张牌。地主为一方，其余两家农民为另一方，双方对战，先出完牌的人所代表的一方获胜；玩法非常简单，牌型变化多，非常考验人的判断和分析能力，游戏对抗性和配合性也很强。

1、发牌

一副牌54张，随机抽一张明牌，然后轮流抓牌。抓到明牌的玩家优先叫地主。最后三张作为游戏的底牌。在没有确定地主之前，底牌不能看。

2、抢地主叫牌

  • 拿到那张明牌的玩家开始叫分，叫牌时可以叫“1分”、“2分”、“3分”或“不叫”。
  • 从后叫分者只能叫比前面玩家高的分或者不叫。
  • 如果有玩家叫“3分”，则立即结束叫牌；如果都不叫，则重新发牌，重新叫分。
  • 在带“抢地主”房间里面，叫完分后，从最后一个叫分的玩家下家开始抢地主。
  • 凡是选择了“不叫”的玩家，没有抢地主的资格。抢地主转到最后一个叫分的玩家结束。
  • 每抢一次地主，则游戏的倍数翻一倍（×2）。如果没人抢地主，则叫分最高的为地主。

3、倍数

  • 在一局刚开始时，倍数为1
  • 如果有人抢地主，则每抢一次地主，游戏的倍数翻一倍（×2）。
  • 如果地主选择了亮牌，则游戏的倍数翻一倍（×2）。
  • 在游戏的过程中，每出一个炸弹或火箭，则游戏的倍数翻一倍（×2）（手上没出的炸弹和火箭不算）。
  • 春天：如果地主出完所有牌，其他两家一张都未出，则游戏的倍数翻一倍（×2）。
  • 反春天：如果其他两家中有一家先出完牌，地主只出过一手牌则游戏的倍数翻一倍（×2）。

4、牌型

火箭 即双王（大王和小王），最大的牌。
炸弹 四张同数值牌（如：7777）。
单牌 单个牌（如：红桃5）。
对牌 数值相同的两张牌（如：梅花4+方块4）。
三张牌 数值相同的三张牌（如：JJJ）。
三带一 数值相同的三张牌 + 一张单牌（如：333+6）。
三带二 数值相同的三张牌 + 一对牌（如：444+99）。
单顺 五张或更多的连续单牌（如：45678或78910JQK）。不包括2点和双王。
连对 三对或更多的连续对牌（如：334455或7788991010JJ）。不包括2点和双王。
三顺 二个或更多的连续三张牌（如：333444或555666777888）。不包括2点和双王。
飞机带翅膀 三顺 + 同数量的单牌或同数量的对牌（如：444555+79或333444555+7799JJ）。
四带二 四张牌 + 两手牌（如：5555＋3＋8或4444＋55＋77）。注意：四带二不是炸弹。

分数算法

  • 地主胜：地主得分为 2×底注×叫分分数×倍数。 其余玩家各得：-底注×叫分分数×倍数。
  • 地主败：地主得分为 -2×底注×叫分分数×倍数。 其余玩家各得：底注×叫分分数×倍数。
  • 如果游戏流局（没人叫地主），则不算任何分数。

癞子场帮助

【牌型】

赖子牌：也称混牌，是可以代替除大小王之外的任何一张牌的全能牌，每副牌共4张。 注意：癞子可以和其他牌搭配成上述各种牌型；但癞子单独出时不能当作万能牌，只能作为其本身使用。例如：癞子为7， 44556 + 7会作为445566的双顺使用；77单出时，只能作为一对7使用。

纯赖子炸弹 四张赖子牌组成的炸弹（例如癞子为5，四个5即为纯赖子炸弹）
火箭 即双王（大王和小王）。
硬炸弹 四张点数相同的牌，如7777。
软炸弹 由癞子搭配成的炸弹，如癞子为7，三个A加7就构成软四个A炸弹，或两个A加两个7也可构成四个A的软炸弹。
单牌 任意一张单牌。
硬对牌 任意两张点数相同的牌。
软对牌 单张牌和赖子构成的对。
硬三张 任意三张点数相同的牌，如888。
软三张 单张加两个赖子或双张加一个赖子
硬三带单张或一对 点数相同的三张牌+一张单牌或一对牌。如333+6 或 444+99。
软三带单张或一对 单张加两个赖子或双张加一个赖子+一张单牌或一对牌。
硬单顺 任意五张或五张以上点数相连的牌如45678或78910JQK。不包括 2和双王。
软单顺 含赖子任意五张或五张以上点数相连的牌
硬双顺 三对或更多的连续对牌，如334455、7788991010JJ。不包括 2 点和双王。
软双顺 含赖子的任意三对或更多的连续对牌
三顺 二个或更多的连续三张牌，如333444 、555666777888。不包括 2 点和双王。
软三顺 含赖子二个或更多的连续三张牌
飞机带翅膀 三顺＋同数量的单牌或同数量的牌。如444555+79 或333444555+7799JJ或666777+55 JJJQQQKKK+334
软飞机带翅膀 含赖子的三顺＋同数量的单牌或同数量的牌。
四带二 四张牌＋一对或者两对或两单牌。（注意：四带二不是炸弹）。如6666+33或5555＋3＋8 或 4444＋55＋77 。
注:AAA222不是飞机（飞机必须是顺子相连的牌）如333444、555666、kkkAAA
软四带二 含癞子的四张牌＋一对或者两对或两单牌。

除炸弹以外牌型，软跟硬同大小牌型是一样大，无大小之分如赖子为4 硬顺子56789 软顺子45679（赖子配8）是一样大

【牌型比较】

火箭>纯赖子炸弹>硬炸弹（没有赖子牌的炸弹）>软炸弹（有赖子牌的炸弹）>一般牌型（单牌、对牌、三张牌、三带一、单顺、双顺、三顺、飞机带翅膀、四带二）。 其中：同等级别的炸弹按牌的分值比大小。

【炸弹积分法】

  • 软炸弹，倍数×2
  • 硬炸弹，倍数×4
  • 火箭，倍数×4
  • 纯癞子炸弹，倍数×64
